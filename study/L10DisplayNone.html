다양한 방법으로 요소를 숨겨봅시다

  • hidden
    hidden 은 input 만 사라짐
  • display : none
    (요소가 없는것 처럼 취급됨)
  • opacity(투명도):0
    (보이지만 않음)
  • 컨텐츠
    요소의 크기를 0으로 만들어도 컨텐츠는 없어지지 않음
  • width:0; overflow:hidden;
    컨텐츠
    넘쳐흐르는 컨텐츠를 hidden으로 감춘다
  • border 가 존재하면 크기로 사라지게 하지 못함
  • box-sizing:border-box; 로 해도 크기가 무관하게 border가 생성 무조건 borber width=0로 지정, border:none
  • ?? 공간은 남아있다
  • transform
    transform 을 이용해 제거 (공간은 남아있다?)
