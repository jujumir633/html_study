  • block 요소는 기본 너비는 브라우저 너비(100%), 컨텐츠가 높이를 결정(auto)
  • html 컨텐츠는 /n 라인개행을 무시
  • block 의 높이를 지정하면 고정크기로 바뀐다. 컨텐츠가 밖으로 나오게된다.(overflow:visible)
  • overflow 컨텐츠가 블럭을 나왔을 때 어떻게 보일지를 결정 (visible,hidden,scroll)
  • block은 박스모델의 contents, padding, border, margin을 갖는다.
  • 박스모델의 contents의 기본값은 border를 포함하지 않는다 (넓이 높이는 컨텐츠의 크기를 지정)
  • box-sizing 은 대부분의 사람이 border까지를 컨텐츠로 인식해서 등장

방향 단축

  • padding: A; → 모든 방향 A
  • padding: A B; → 위아래 A, 좌우 B
  • padding: A B C; → 위 A, 좌우 B, 아래 C
  • padding: A B C D; → 위 A, 오른쪽 B, 아래 C, 왼쪽 D (시계방향)
