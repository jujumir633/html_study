CSS의 단위에 대해 알아 봅시다.

cm, m, km, 원, 도씨와 같은 일상적인 단위는 css에서 존재하지 않는다

브라우저(css)에서 사용하는 단위들

  • px,% :
    30%*30%
    부모요소의 크기를 기준으로 자식의 비유율 정의
  • vw, vh(viewport 브라우저 100~0) : 커졌다 작아졌다[모바일 크기]
    10vw*10vh
  • rem (1rem==16px(기본font size)) : 글자크기를 기준으로 크기지정()[모바일크기]
    16px*16px
  • em(시작은 16px==1em, 부모요소의 크기를 기준으로 계속 변경됨) 부모가1em(16px) 이고 자식이2em이면 부모 16px x2em 해서 32px가 됨
    10em x 10em

색에서 사용하는 단위

  • rgb(0~255, 0~255, 0~255) :
    rgb(255,0,0)
  • rgba(0~255, 0~255, 0~255, 0~1) :
    rgb(255,0,0,0.5)
  • 16진수 색 #(00~FF == 0~255, 00~FF) :
    머고
  • HSL (색조 Hue, 채도,명도)
    첫번째
  • HSLA [색조 Hue, 채도(0~100),명도(0~100),알파 투명도(0~1)]
    두번째

단위 작성 시 주의할점

  • 0을 제외하고 단위가 없는 수는 없다
  • 1000x1000
    0을 제외하고 단위가 없는 수는 없다
  • 100px+10px
    계산으로 값을 대입할 수 없다
  • calc(300px/4)
    계산할때는 calc 사용
  • calc(300px+100px)
