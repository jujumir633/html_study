조건문 if, else, switch

조건문(Conditional Statement)이란?

  • 조건문은 프로그램이 특정 조건을 만족할 때만 실행되는 코드를 작성하는 방법
  • 조건에 따라 실행되는 코드가 달라지므로, 프로그램에 의사결정을 부여할 수 있음

조건문을 사용하는 목적

  • 상황에 따라 다른 동작을 하도록 만들기 위해 사용
  • 예: 나이가 19세 이상이면 "성인" 출력, 아니면 "미성년자" 출력
  • 즉, 프로그램의 흐름을 제어하고 논리적인 분기를 만들 수 있음

if 조건문 작성법

1. 기본 구조

if (조건식) {
  // 조건이 true일 때 실행되는 코드
}

2. if ~ else

if (조건식) {
  // 조건이 true일 때 실행
} else {
  // 조건이 false일 때 실행
}

3. if ~ else if ~ else

if (조건식1) {
  // 조건1이 true일 때 실행
} else if (조건식2) {
  // 조건1은 false, 조건2가 true일 때 실행
} else {
  // 모든 조건이 false일 때 실행
}

조건문 예시

  • 나이 판별: 
    let age = 20;
if (age >= 19) {
  console.log("성인입니다.");
} else {
  console.log("미성년자입니다.");
}
  • 점수 판별: 
    let score = 85;
if (score >= 90) {
  console.log("A 학점");
} else if (score >= 80) {
  console.log("B 학점");
} else {
  console.log("C 학점 이하");
}

3항 연산자(Ternary Operator)란?

  • 조건식에 따라 두 가지 결과 중 하나를 선택하는 연산자
  • if ~ else를 짧게 표현할 때 사용
  • 형식: 조건식 ? 값1 : 값2
  • 조건식이 true면 값1, false면 값2를 반환
  • 간단한 분기 처리를 한 줄로 작성할 수 있음
  • 조건문을 변수에 바로 대입할 때 유용

예시

1. 기본 예시

let age = 20;
let result = (age >= 19) ? "성인" : "미성년자";
console.log(result); // "성인"

2. if ~ else 와 비교

// if ~ else
let message;
if (age >= 19) {
  message = "성인";
} else {
  message = "미성년자";
}

// 3항 연산자
let message2 = (age >= 19) ? "성인" : "미성년자";

3. 중첩 사용 (주의 필요)

let score = 85;
let grade = (score >= 90) ? "A"
            : (score >= 80) ? "B"
            : "C";
console.log(grade); // "B"

switch 문이란?

  • 하나의 변수나 표현식 값을 여러 경우(case)와 비교하여 실행할 코드를 선택하는 조건문
  • if ~ else if 문을 간결하게 표현할 때 사용

switch 문 기본 구조

switch(표현식) {
  case 값1:
    // 표현식 === 값1 일 때 실행되는 코드
    break;
  case 값2:
    // 표현식 === 값2 일 때 실행되는 코드
    break;
  default:
    // 위 조건이 모두 해당되지 않을 때 실행
}

예시

1. 요일 출력

let day = 2;
let dayName;

switch(day) {
  case 0:
    dayName = "일요일";
    break;
  case 1:
    dayName = "월요일";
    break;
  case 2:
    dayName = "화요일";
    break;
  default:
    dayName = "기타 요일";
}

console.log(dayName); // "화요일"

2. break가 없을 때 (fall-through)

let fruit = "사과";

switch(fruit) {
  case "사과":
    console.log("사과입니다.");
  case "바나나":
    console.log("바나나입니다.");
}
// 출력: "사과입니다." 후 이어서 "바나나입니다."

switch 문 특징

  • switch는 비교할 값이 많을 때 유용
  • === (엄격한 비교) 방식으로 동작
  • break를 쓰지 않으면 다음 case로 넘어가는 "fall-through" 발생
  • default는 선택사항이며, 모든 case에 해당하지 않을 때 실행

if 문 vs switch 문 비교

평가방식

  • if: 임의의 조건식(범위, 조합, 함수 호출 등)을 평가해 분기
  • switch: 하나의 표현식 값을 여러 상수 케이스와 엄격 비교(===)하여 분기
    • 자바스크립트: 숫자, 문자열, 불린, null, undefined 등 모든 기본형 가능. 객체도 가능하지만 참조 동일성 비교라 거의 사용하지 않음
    • 자바: 기본형(byte, short, char, int) + enum + (Java 7부터) 문자열 가능. 그 외 객체 타입은 불가

가독성

  • if: 범위/복합 조건에 읽기 쉬움 (예: 0 ≤ x < 10, x%2===0 && y>0)
  • switch: 값에 따른 단일 선택이 많을 때 표 형태로 명확 (케이스 5~10개 이상일 때 유리)
  • 중첩/장문 조건은 if가, 다수의 상수 분기는 switch가 일반적으로 더 읽기 좋음

성능(속도) : 차이없음

  • 현대 엔진(JIT)은 두 문법 모두 매우 빠르게 최적화됨 — 대부분의 앱에서 체감 차이 거의 없음
  • **옛날 언어(C, C++, Java 초기)**에서는 컴파일러가에서는 switch가, if-else보다 빠름

언제 무엇을 쓸까

  • switch 사용: 상태코, 명령어, 토큰 등 상수 분기가 많을 때
  • if 사용: 범위 비교, 다중 논리 연산(AND/OR/NOT) 등 복잡한 평가가 필요할 때

간단 예시

  • switch가 더 읽히는 경우 
    switch(command) {
  case "start": start(); break;
  case "stop":  stop();  break;
  case "pause": pause(); break;
  default:      help();
}
  • if가 더 읽히는 경우 
    if (score >= 90) grade = "A";
else if (score >= 80) grade = "B";
else if (score >= 70) grade = "C";
else grade = "F";

조건문(if, else if, else / switch) 연습 문제

문제 1. 점수 등급 출력

  • 사용자로부터 점수를 입력받아 등급을 출력하라.
  • 90점 이상: "A", 80점 이상: "B", 70점 이상: "C", 그 외: "F"
  • if ~ else if ~ else 또는 switch 문으로 작성해볼 것

문제 2. 요일 이름 출력

  • 요일 번호(0~6)를 입력받아 요일 이름을 출력하라.
  • 0: 일요일, 1: 월요일, ..., 6: 토요일
  • switch 문으로 풀면 직관적이고, if 문으로도 풀어볼 것

문제 3. 사칙연산 계산기

  • 두 수와 연산자(+, -, *, /)를 입력받아 결과를 출력하라.
  • switch 문으로 연산자 분기 처리
  • if 문으로도 동일하게 구현해보기

문제 4. 달의 일수

  • 달 번호(1~12)를 입력받아 해당 달의 일수를 출력하라.
  • 2월: 28일 (윤년은 고려하지 않음)
  • 4, 6, 9, 11월: 30일
  • 그 외: 31일
  • switch 문으로 작성해보고, if 문으로도 비교해보기

문제 5. 짝수/홀수 판별

  • 정수를 입력받아 "짝수" 또는 "홀수"를 출력하라.
  • if 문으로 작성해보고, switch 문으로도 가능하도록 도전

조건문 연습 문제 정답

문제 1. 점수 등급 출력

// if ~ else if ~ else
let score = 85;
if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else if (score >= 70) {
  console.log("C");
} else {
  console.log("F");
}

// switch (범위 비교는 switch에 부적합, 변형 예시)
let grade;
switch (true) {
  case (score >= 90): grade = "A"; break;
  case (score >= 80): grade = "B"; break;
  case (score >= 70): grade = "C"; break;
  default: grade = "F";
}
console.log(grade);

문제 2. 요일 이름 출력

// if ~ else
let day = 2;
if (day === 0) console.log("일요일");
else if (day === 1) console.log("월요일");
else if (day === 2) console.log("화요일");
else if (day === 3) console.log("수요일");
else if (day === 4) console.log("목요일");
else if (day === 5) console.log("금요일");
else if (day === 6) console.log("토요일");
else console.log("잘못된 입력");

// switch
switch(day) {
  case 0: console.log("일요일"); break;
  case 1: console.log("월요일"); break;
  case 2: console.log("화요일"); break;
  case 3: console.log("수요일"); break;
  case 4: console.log("목요일"); break;
  case 5: console.log("금요일"); break;
  case 6: console.log("토요일"); break;
  default: console.log("잘못된 입력");
}

문제 3. 사칙연산 계산기

let a = 10, b = 5, op = "*";

// if ~ else
if (op === "+") console.log(a + b);
else if (op === "-") console.log(a - b);
else if (op === "*") console.log(a * b);
else if (op === "/") console.log(a / b);
else console.log("지원하지 않는 연산자");

// switch
switch(op) {
  case "+": console.log(a + b); break;
  case "-": console.log(a - b); break;
  case "*": console.log(a * b); break;
  case "/": console.log(a / b); break;
  default: console.log("지원하지 않는 연산자");
}

문제 4. 달의 일수

let month = 4;

// if ~ else
if (month === 2) {
  console.log("28일");
} else if (month === 4 || month === 6 || month === 9 || month === 11) {
  console.log("30일");
} else if (month >= 1 && month <= 12) {
  console.log("31일");
} else {
  console.log("잘못된 달 번호");
}

// switch
switch(month) {
  case 2:
    console.log("28일");
    break;
  case 4: case 6: case 9: case 11:
    console.log("30일");
    break;
  case 1: case 3: case 5: case 7: case 8: case 10: case 12:
    console.log("31일");
    break;
  default:
    console.log("잘못된 달 번호");
}

문제 5. 짝수/홀수 판별

let num = 7;

// if ~ else
if (num % 2 === 0) console.log("짝수");
else console.log("홀수");

// switch
switch(num % 2) {
  case 0: console.log("짝수"); break;
  case 1: console.log("홀수"); break;
  default: console.log("숫자가 아님");
}
