JavaScript 반복자와 Array 반복문

1. 반복자(Iterator)

  • 반복자는 next()를 사용해 값을 하나씩 꺼낼 수 있는 객체이다.
  • { value: 값, done: 불린 } 형태를 반환한다.
  • 배열, 문자열, Map, Set 등은 기본적으로 Symbol.iterator를 제공한다.
const arr = [10, 20, 30];
const it = arr[Symbol.iterator]();

console.log(it.next()); // { value: 10, done: false }
console.log(it.next()); // { value: 20, done: false }
console.log(it.next()); // { value: 30, done: false }
console.log(it.next()); // { value: undefined, done: true }

2. 기본 반복문

  • for: 인덱스를 직접 제어
  • for...of: 값을 순회(Iterable 기반)
  • for...in: 키/인덱스 순회(배열보다는 객체에 사용 권장)
  • forEach: 각 요소에 대해 콜백 실행(반환 없음)
const fruits = ["사과", "바나나", "포도"];

for (let i = 0; i < fruits.length; i++) {
  console.log(i, fruits[i]);
}

for (const f of fruits) {
  console.log(f);
}

for (const idx in fruits) {
  console.log(idx, fruits[idx]);
}

fruits.forEach((f, i) => console.log(i, f));

3. Array 메서드와 체이닝

  • map((item, index, arr) => {}): 요소 변환 → 새 배열 반환
  • filter((item, index, arr) => {}): 조건 만족 요소만 반환
  • reduce((acc, item, index, arr) => {}, 초기값): 누적 계산
  • some((item, index, arr) => {}) / every((item, index, arr) => {}): 조건 존재 여부 확인
  • find((item, index, arr) => {}) / findIndex((item, index, arr) => {}): 첫 일치 요소/인덱스 반환
const nums = [1,2,3,4,5,6];

// 체이닝 예시: 짝수 → 2배 → 합계
const sum = nums
  .filter(n => n % 2 === 0)   // [2,4,6]
  .map(n => n * 2)            // [4,8,12]
  .reduce((acc, cur) => acc + cur, 0);

console.log(sum); // 24

// 문자열 다듬기 체이닝
const words = ["  apple ", "banana", " pear  "];
const result = words
  .map(w => w.trim())
  .map(w => w.toLowerCase())
  .map(w => w.length);

console.log(result); // [5,6,4]

4. 응용 예시

const users = [
  { name: "Kim", score: 88 },
  { name: "Lee", score: 72 },
  { name: "Park", score: 95 },
  { name: "Choi", score: 91 },
];

// 상위 3명 이름만 추출
const top3 = users
  .slice() // 원본 보호
  .sort((a,b) => b.score - a.score)
  .slice(0,3)
  .map(u => u.name);

console.log(top3); // ["Park", "Choi", "Kim"]
