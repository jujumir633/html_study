반복문 for while

반복문(Loop)이란?

  • 반복문은 특정 코드 블록을 여러 번 실행하기 위해 사용
  • 조건이 true인 동안 반복하거나, 정해진 횟수만큼 반복 가능
  • 프로그램에서 같은 작업을 효율적으로 처리할 수 있게 함

반복문을 사용하는 목적

  • 동일한 코드를 여러 번 작성하지 않고, 한 번의 정의로 반복 실행
  • 대량의 데이터 처리(예: 배열의 모든 값 출력)
  • 조건을 만족할 때까지 작업 반복

반복문의 종류

1. for 문

  • 반복 횟수가 정해져 있을 때 사용
  • 형식: for (초기식; 조건식; 증감식) { ... }
for (let i = 0; i < 5; i++) {
  console.log(i); // 0,1,2,3,4
}

2. while 문

  • 조건이 true인 동안 반복
  • 반복 횟수를 알 수 없을 때 사용
let n = 0;
while (n < 3) {
  console.log(n);
  n++;
}

3. do ~ while 문

  • 조건을 나중에 검사 → 최소 한 번은 실행
let x = 0;
do {
  console.log(x);
  x++;
} while (x < 3);

4. for ... of 문

  • 배열, 문자열 등 iterable 객체의 값을 순회
let fruits = ["사과", "바나나", "체리"];
for (let fruit of fruits) {
  console.log(fruit);
}

5. for ... in 문

  • 객체의 키(속성 이름)을 순회
let user = { name: "경민", age: 39 };
for (let key in user) {
  console.log(key, ":", user[key]);
}

반복문 제어

  • break: 반복 즉시 종료
  • continue: 현재 반복 건너뛰고 다음 반복 진행
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // 3 건너뜀
  if (i === 5) break;    // 5에서 종료
  console.log(i); // 1,2,4 출력
}
