구현 방법 정리

  • 프로젝트 구조: index.html 하나에 네비게이션, 툴바, 카드 그리드, 페이지네이션 순서로 배치
  • 메타와 CSS: <meta name="viewport" content="width=device-width, initial-scale=1"> 포함, Bootstrap 5 CSS CDN 사용
  • 그리드 컨테이너: <main><div class="container"></div></main> 구조, 내부는 .row로 시작
  • 반응형 열 배치: 각 카드 래퍼에 col-12 col-md-6 col-lg-3 d-flex 클래스 사용
    • 모바일: col-12 → 1열
    • 태블릿: col-md-6 → 2열
    • 데스크톱: col-lg-3 → 4열
  • 카드 높이 균일화: 카드에 card h-100 w-100, 본문은 card-body d-flex flex-column 사용, 요약 텍스트에 flex-grow-1 적용
  • 썸네일 비율 유지: .ratio .ratio-16x9 안에 img 넣고 object-fit-cover 적용
  • 카드 메타 정보: 작성자, 날짜, 조회수는 p.text-muted.small.d-flex.gap-3 형태로 배치
  • 카드 하단: 왼쪽은 badge 그룹, 오른쪽은 btn.btn-sm.btn-primary 배치
  • 툴바: 검색은 input-group, 정렬/카테고리는 select.form-select, 카테고리 Pills는 ul.nav.nav-pills
  • 페이지네이션: ul.pagination.justify-content-center 구조 사용
  • 브레이크포인트 요약
    • col-* : <576px
    • col-md-* : ≥768px
    • col-lg-* : ≥992px
  • 접근성: 모든 입력에 <label> 연결, 이미지에는 alt 속성 추가, 네비게이션에 aria-label 사용
  • 자주 생기는 문제 해결
    • 반응형이 안 바뀜 → viewport 메타 확인
    • 카드 높이 들쑥날쑥 → h-100 + d-flex + flex-grow-1
    • 이미지 찌그러짐 → ratio + object-fit-cover
    • 열이 4개 안 됨 → .col-lg-3 누락 여부 확인
