인라인에 블럭을 추가하는 float

float

  • 개념 요소를 문서의 일반적인 흐름에서 분리해 좌측이나 우측으로 배치하고, 다른 인라인 요소들이 그 옆을 감싸도록 하는 CSS 속성. 원래는 이미지 옆에 텍스트를 흐르게 하기 위해 만들어졌지만, 과거에는 레이아웃 제작에도 많이 사용됨.
  • 속성 값 - none : 기본값, float 적용 안 함 - left : 요소를 왼쪽으로 띄움 - right : 요소를 오른쪽으로 띄움 - inherit : 부모의 float 값 상속
  • 해제 방법(clear) float된 요소의 영향을 받지 않게 하려면 clear 속성 사용 - clear: left; : 왼쪽 float 해제 - clear: right; : 오른쪽 float 해제 - clear: both; : 양쪽 float 해제
  • 문제점과 해결책 부모 요소가 float 자식의 높이를 인식하지 못하는 문제 발생. 해결 방법: - 부모에 overflow: hidden; 또는 overflow: auto; 적용 - CSS clearfix 기법 사용 (가상 요소 ::after로 clear 적용) - 레이아웃 목적이라면 float 대신 flexbox나 grid 사용 권장

clear를 사용하는 이유

  • float 영향 제거 float된 요소는 문서의 일반적인 흐름에서 벗어나 좌우에 띄워지므로, 뒤따르는 요소가 그 옆에 배치되거나 겹칠 수 있습니다. clear는 해당 방향의 float 요소 옆에 오지 않게 합니다.
  • 레이아웃 무너짐 방지 부모 안의 자식 요소가 모두 float이면, 부모가 자식의 높이를 인식하지 못해 영역이 0이 됩니다. clear를 사용하면 다음 요소가 float 아래로 내려가면서 부모 영역이 정상적으로 잡힙니다.
  • 가독성과 구조 유지 float된 요소와 그 다음 요소 사이의 시각적·구조적 구분을 명확히 하여, 레이아웃이 의도한 대로 유지됩니다.
  • 방향 지정 가능 clear: left, clear: right, clear: both를 통해 어느 방향의 float을 피할지 명확히 설정할 수 있습니다.
