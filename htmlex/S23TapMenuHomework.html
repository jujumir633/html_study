과제: 순수 CSS 탭 메뉴 구현하기

  • 목표 JavaScript 없이 HTML과 CSS만 사용하여 탭 메뉴를 구현한다. 각 탭을 클릭하면 해당 콘텐츠가 부드럽게 펼쳐지고, 다른 콘텐츠는 접히는 형태로 동작해야 한다.
  • 구현 방식
    • 라디오 버튼 + label을 이용하여 탭을 전환한다. label을 클릭하면 연결된 input[type="radio"]가 체크되며, 이를 CSS로 감지해 콘텐츠를 토글한다.
    • :has() 선택자를 활용해 “특정 라디오 버튼이 체크되었을 때 해당 콘텐츠를 보이게” 만든다.
    • 애니메이션은 display:none 대신 max-height, opacity, transform을 조합해 부드럽게 펼침/접힘 효과를 만든다.
  • :has() 선택자 설명
    • :has()는 부모나 조상 요소가 특정 자식/후손을 포함했을 때 스타일을 적용할 수 있는 관계 의사 클래스다.
    • 예: .container:has(#tab1:checked) #content1 { ... } → .container 안에서 #tab1 라디오 버튼이 체크되면 #content1에 스타일이 적용된다.
    • 이를 통해 구조를 크게 바꾸지 않고도 CSS만으로 상태 기반 UI를 만들 수 있다.
  • 세부 요구사항
    • 탭은 최소 3개 이상 구성한다(예: 홈, 소개, 연락).
    • 각 탭의 콘텐츠는 접힌 상태에서 보이지 않다가 선택된 탭일 때만 펼쳐진다.
    • 전환 시 애니메이션이 적용되어 자연스럽게 열리고 닫혀야 한다.
    • 탭 버튼에 hover 시 배경색 변화 같은 시각적 효과를 준다.
    • 전체 레이아웃은 중앙 정렬되며, 카드 형태의 디자인(둥근 모서리, 그림자 효과 등)을 적용한다.
