CSS와 스타일 속성 정리

CSS란?

  • CSS(Cascading Style Sheets)는 HTML 요소의 디자인(모양과 배치)을 지정하는 언어입니다.
  • HTML은 구조, CSS는 표현을 담당합니다.
  • 하나의 CSS 파일로 여러 HTML 문서에 스타일을 적용할 수 있습니다.

스타일 속성이란?

  • HTML 태그 안에 style 속성을 사용해 인라인으로 CSS를 지정할 수 있습니다.
  • 한 요소에 한 번만 적용되며, 우선순위가 높습니다.
  • 예시: <p style="color:red;">빨간 글씨</p>

자주 쓰는 CSS 속성

  • color : 텍스트 색상
  • background-color : 배경색
  • font-size : 글자 크기
  • text-align : 텍스트 정렬 (left, center, right)
  • width, height : 요소의 너비와 높이
  • padding : 안쪽 여백
  • margin : 바깥 여백
  • border : 테두리

CSS 선택자 정리

  • 전체 선택자 (*) 모든 요소에 스타일 적용
    * { margin: 0; }
  • 태그 선택자 특정 태그에 스타일 적용
    p { color: blue; }
  • 클래스 선택자 (.) 특정 class 속성을 가진 요소에 적용
    .highlight { background-color: yellow; }
  • ID 선택자 (#) 특정 id 속성을 가진 요소에 적용
    #main-title { font-size: 24px; }
  • 그룹 선택자 여러 선택자를 콤마로 묶어 동시에 지정
    h1, h2, p { font-family: Arial; }
  • 자식 선택자 (>) 바로 아래 자식 요소만 선택
    ul > li { color: red; }
  • 후손 선택자 (띄어쓰기) 하위 모든 자식 요소 선택
    div span { font-weight: bold; }
  • 인접 형제 선택자 (+) 바로 뒤에 있는 형제 요소 하나 선택
    h1 + p { color: gray; }
  • 일반 형제 선택자 (~) 뒤에 있는 모든 형제 요소 선택
    h1 ~ p { font-style: italic; }
  • 속성 선택자 특정 속성을 가진 요소 선택
    input[type="text"] { border: 1px solid #ccc; }

CSS 선택자 우선순위

동일한 요소에 여러 스타일이 적용될 때 어떤 스타일이 우선 적용되는지 결정하는 규칙입니다.

우선순위 순서 (낮음 → 높음)

  • * (전체 선택자)
  • 태그 선택자 (예: div, p)
  • 클래스 선택자, 속성 선택자, 가상 클래스 (예: .class, [type="text"], :hover)
  • ID 선택자 (예: #id)
  • 인라인 스타일 (예: style="...")
  • !important (가장 강력한 우선순위)

우선순위 점수 개념 (간단)

  • 인라인 스타일: 1000점
  • ID 선택자: 100점
  • 클래스, 속성, 가상 클래스: 10점
  • 태그 선택자, 가상 요소: 1점

예시

아래 두 스타일 중 어떤 것이 적용될까요?


p {
  color: blue;  /* 태그 선택자 */
}
#myPara {
  color: red;   /* ID 선택자 */
}

→ #myPara가 더 높은 우선순위를 가져 텍스트가 빨간색으로 표시됩니다.

HTML & CSS 과제

1. HTML 요소와 속성 작성하기

  • ul 리스트를 만들고, 그 안에 3개의 li 요소를 작성하세요.
  • 각 li에 title 속성을 추가하여 마우스를 올리면 설명이 나타나게 하세요.
  • li 중 하나에 class="highlight"를 부여하세요.

2. CSS 스타일 적용하기

  • .highlight 클래스에 background-color: yellow;, color: red;를 적용하세요.
  • 모든 li 요소에 text-align: center;를 적용하세요.
  • ul에 padding: 20px;, margin: 30px;를 지정하세요.

3. CSS 선택자와 우선순위 연습

  • id="main-list"를 ul에 추가하고, #main-list 선택자로 border: 2px solid blue;를 적용하세요.
  • li 태그 선택자로 font-size: 18px;를 지정하세요.
  • 특정 li에 style 속성을 사용해 color: green;를 직접 지정해 보세요.

4. 예쁜 프로필 카드 만들기 (구현 문제)

  • 이름 (큰 글씨, 굵게)

  • 연락처 또는 이메일 (작고 회색 글씨)

  • 사진은 둥근 원형으로 만들기
  • 카드에 적당한 안쪽 여백(padding)과 바깥 여백(margin) 적용
