요소를 감추는 방법

display: none

  • 요소 자체가 화면에서 완전히 제거됨
  • 문서 흐름에서도 사라짐 (자리도 없어짐)
  • 스크린 리더도 인식 못함 (접근성 주의)

visibility: hidden

  • 요소는 보이지 않지만, **자리는 유지**됨
  • 문서 흐름에 영향 없음
  • 스크린 리더는 인식할 수 있음

opacity: 0

  • 요소가 완전히 투명해져 보이지 않음
  • 자리는 유지됨
  • 스크린 리더는 인식함
  • 마우스 이벤트는 그대로 감지됨 (주의!)

width: 0 + overflow: hidden

  • 요소의 크기를 0으로 줄이고 내부 내용도 숨김
  • 공간도 줄고 컨텐츠도 보이지 않음
  • border가 있다면 눈에 보일 수 있으므로 border: none 권장

transform: scale(0)

  • 요소 크기를 시각적으로 0으로 축소
  • 자리(공간)는 유지되며, 접근성도 그대로 있음
  • 애니메이션용으로 많이 사용됨

HTML 속성: hidden

  • <element hidden>처럼 HTML 속성으로 숨김
  • CSS의 display: none과 동일한 효과
  • JavaScript로 element.hidden = true 사용 가능
  • input[type="hidden"]은 폼 제출용 요소로, 시각적으로는 원래 안 보임
