CSS 단위 정리

📏 크기 단위

  • px – 픽셀 (기본 고정 단위)
  • % – 부모 요소 기준 비율
  • em – 부모 폰트 크기 기준 (상대 단위)
  • rem – 루트 폰트 크기 기준 (HTML 기본 16px)
  • vw / vh – 뷰포트 너비 / 높이 기준 (반응형에서 필수)
  • vmin / vmax – 뷰포트의 작은 / 큰 축 기준
  • calc() – 여러 단위를 계산해서 조합할 때 사용

🎨 색상 단위

  • rgb(r, g, b) – 정수값 (0~255)
  • rgba(r, g, b, a) – 투명도 포함
  • #rrggbb – 16진수 표현
  • hsl(h, s%, l%) – 색조, 채도, 명도
  • hsla(h, s%, l%, a) – 투명도 포함 HSL

크기 단위 예시

  • px (픽셀): 고정된 단위, 화면 해상도 기준
    100px
  • % (백분율): 부모 요소 기준
    50%
  • vw / vh (뷰포트 너비 / 높이 기준):
    20vw / 20vh
  • rem (루트 폰트 크기 기준, 기본 1rem = 16px)
    2rem
  • em (부모 폰트 크기 기준)
    10em

색상 단위 예시

  • rgb(r, g, b): 정수값(0~255)
    RGB
  • rgba(r, g, b, a): RGB + 투명도(0~1)
    RGBA
  • #RRGGBB (16진수 표기법)
    #FF0000
  • hsl(H, S%, L%): 색상(0~360), 채도/명도(0~100%)
    HSL
  • hsla(H, S%, L%, A): HSL + 투명도
    HSLA

단위 사용 시 주의사항

  • 숫자 0은 단위 없이 사용 가능 (ex. `width: 0;`)
    0
  • 0이 아닌 숫자에는 반드시 단위가 필요함
    잘못된 예
  • 단위끼리 수학 연산은 직접 불가능
    오류
  • 계산할 경우 `calc()` 함수 사용
    calc
  • `calc()` 연산 시 연산자 주변에 공백 필수
    OK
