한줄 레이아웃을 구성하는 Flex

Flexbox (Flexible Box Layout)

  • 개념 CSS 레이아웃 방식으로, 요소들을 한 방향(가로 또는 세로)으로 배치하고 정렬하기 쉽게 만들어줍니다.
  • 장점
    • 간단한 코드로 가로/세로 중앙 정렬 가능
    • 요소 간 간격 조절 및 균등 배치 용이
    • 반응형 레이아웃 구현에 최적화
    • float나 inline-block 사용 시 발생하는 문제점 해결 가능
  • 주요 개념
    • flex-direction이 주축 방향을 결정
    • justify-content는 주축 정렬, align-items는 교차축 정렬 담당
    • 주축(main axis)과 교차축(cross axis) 개념 이해가 중요
  • Flex Container - 주요 속성
    • display: flex; 또는 inline-flex;를 선언하면 해당 요소가 Flex Container가 됩니다.
    • flex-direction: 주축 방향 설정
      • row (기본값): 가로 방향
      • row-reverse: 가로 반대 방향
      • column: 세로 방향
      • column-reverse: 세로 반대 방향
    • flex-wrap: 줄바꿈 여부
      • nowrap (기본값): 한 줄에 배치
      • wrap: 공간 부족 시 줄바꿈
      • wrap-reverse: 줄바꿈 + 반대 방향
    • justify-content: 주축 정렬
      • flex-start (기본값)
      • flex-end
      • center
      • space-between
      • space-around
      • space-evenly
    • align-items: 교차축 정렬
      • stretch (기본값)
      • flex-start
      • flex-end
      • center
      • baseline
    • align-content: 여러 줄 교차축 정렬(flex-wrap: nowrap)
      • stretch (기본값)
      • flex-start
      • flex-end
      • center
      • space-between
      • space-around
  • Flex Item - 주요 속성
    • flex container 의 자식요소는 모두 flex item으로 취급됩니다.
    • order: 아이템 순서 변경
    • flex-grow: 남는 공간을 얼마나 차지할지 비율 지정
    • flex-shrink: 공간 부족 시 축소 비율 지정
    • flex-basis: 기본 크기 지정
    • flex: grow, shrink, basis를 한번에 설정하는 축약 속성
    • align-self: 개별 아이템 교차축 정렬 조절
