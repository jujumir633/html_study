position 으로 요소의 위치 지정

position: static (기본값)

  • 기본 배치 상태이며 따로 위치 지정하지 않음
  • top, left 등의 위치 속성이 무시됨

position: relative (상대 위치)

  • 요소의 원래 위치(static 기준)를 기준으로 이동
  • 문서 흐름에는 영향을 주지 않음 (자리 차지 유지)
  • 자식의 absolute 위치 기준이 됨

position: absolute (절대 위치)

  • 가장 가까운 relative, absolute, fixed, sticky 부모 기준으로 위치함
  • 문서 흐름에서 제거되어 겹칠 수 있음
  • 기준 부모가 없으면 <html> 기준

position: fixed (뷰포트 기준 고정)

  • 브라우저 화면(뷰포트)을 기준으로 항상 고정
  • 스크롤해도 같은 위치에 유지됨
  • 주로 헤더/사이드바/알림창 등에 사용

position: sticky (스크롤에 따라 고정)

  • 스크롤 시 지정 위치에 도달하면 고정됨
  • 그 이전에는 relative처럼 동작
  • 부모 요소의 영역을 벗어나면 다시 스크롤됨
  • top, left 등의 기준값 필요

top / right / bottom / left

  • 요소의 위치를 조절하는 속성
  • relative, absolute, fixed, sticky에서만 동작
  • 위치를 지정할 축 기준은 부모 또는 뷰포트에 따라 다름
  • 예: top: 10px → 위에서 10px 아래로 이동

z-index (요소의 쌓이는 순서)

  • 요소가 화면에 **겹칠 때 위/아래 순서**를 결정
  • 값이 클수록 위에 표시됨
  • 음수도 가능 (z-index: -1 → 맨 뒤)
  • position이 지정된 요소에서만 동작 (relative, absolute, fixed, sticky)
