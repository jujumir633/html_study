CSS 의사 요소 선택자

  • ::before - 요소의 내용 앞에 가상의 콘텐츠를 추가
  • ::after - 요소의 내용 뒤에 가상의 콘텐츠를 추가
  • ::first-letter - 첫 번째 글자에 스타일 적용
  • ::first-line - 첫 번째 줄에 스타일 적용
  • ::selection - 사용자가 선택한 텍스트에 스타일 적용
  • ::marker - 목록 불릿이나 번호에 스타일 적용

사용 예시

  • p::before { content: "★"; color: gold; } - 문단 앞에 별 추가
  • p::after { content: " (끝)"; color: gray; } - 문단 뒤에 '(끝)' 추가
  • p::first-letter { font-size: 2em; } - 첫 글자 확대
  • p::first-line { color: blue; } - 첫 줄 색상 변경
  • ::selection { background: yellow; } - 선택 영역 배경 노랑
  • li::marker { color: red; } - 리스트 불릿 빨강

실습 예제

주의사항

  • ::before, ::after는 반드시 content 속성을 지정해야 표시됨
  • CSS3부터는 :: (더블 콜론) 표기를 권장
  • 의사 요소는 HTML에 없는 가상의 요소에 스타일 적용

1. ::before로 꾸미기(문제 시작)

  • 목표: 모든 <h2> 제목 앞에 "★ " 아이콘을 가상으로 추가
  • 조건: 실제 HTML에는 추가하지 않고 CSS로만 구현
  • 키워드: ::before, content
  • 주의 포인트: ::before는 content 속성이 반드시 필요

2. ::after로 꾸미기

  • 목표: 버튼 뒤에 " →" 화살표 모양을 가상으로 추가
  • 조건: hover 상태일 때 화살표 색이 파란색으로 변경
  • 키워드: ::after, :hover, content
  • 주의 포인트: 의사 요소와 의사 클래스를 함께 적용 가능

3. ::first-letter 스타일링

  • 목표: 모든 단락(<p>)의 첫 글자를 2배 크기, 빨간색, 굵게 표시
  • 조건: 나머지 글자는 스타일이 변하지 않아야 함
  • 키워드: ::first-letter
  • 주의 포인트: 블록 요소에서만 적용 가능

4. ::first-line 스타일링

  • 목표: 모든 단락의 첫 줄만 배경색 노랑, 글자색 파랑
  • 조건: 글자 수를 늘려서 줄바꿈이 생기도록 테스트
  • 키워드: ::first-line
  • 주의 포인트: 줄 수가 바뀌면 스타일 적용 범위도 달라짐

5. ::selection 스타일링

  • 목표: 사용자가 텍스트를 드래그 선택했을 때 배경색을 검정, 글자색을 노랑으로 표시
  • 조건: 단락 텍스트에만 적용
  • 키워드: ::selection
  • 주의 포인트: 선택 영역 스타일은 브라우저별 지원 차이가 있을 수 있음

6. 의사 클래스 + 의사 요소 조합(문제 끝)

  • 목표: 목록 항목(<li>)에 마우스를 올렸을 때, 항목 앞에 "👉 " 가상 요소 추가
  • 조건: 마우스를 치우면 가상 요소가 사라짐
  • 키워드: :hover, ::before, content
  • 주의 포인트: :hover와 ::before를 함께 사용할 수 있음
