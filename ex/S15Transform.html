1) 브라우저 렌더링 과정

1. DOM/CSSOM 생성 : HTML과 CSS를 파싱해 구조와 스타일 트리를 만든다.
2. 레이아웃(Layout/Reflow) : 각 요소의 크기와 위치를 계산한다.
3. 페인트(Paint) : 색, 텍스트, 이미지 등을 화면에 그릴 픽셀 데이터로 만든다.
4. 합성(Compositing) : 여러 레이어를 GPU에서 합쳐 최종 화면을 만든다.

2) transform과 일반 스타일 변경의 차이

레이아웃 속성 변경 (예: left, top, width, height)은 레이아웃과 페인트 단계를 다시 거쳐야 하므로 성능 부담이 크다.
transform은 요소의 시각적인 위치·크기·회전만 합성 단계에서 변경하므로 주변 요소 레이아웃에 영향을 주지 않고 비교적 부드럽다.
