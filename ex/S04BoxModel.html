📦 CSS 박스 모델(Box Model) 정리

1. 박스 모델이란?

  • HTML 요소는 기본적으로 박스 형태로 구성되며, 각 요소는 다음과 같은 구조를 가짐
  • 구성 요소 (안쪽부터 바깥쪽 순서): content → padding → border → margin
  • 박스 모델은 요소의 전체 크기를 결정하는 중요한 기준

2.box 모델의 구성 요소 설명

  • Content: 요소의 실제 콘텐츠 영역 (텍스트, 이미지 등)
  • Padding: 콘텐츠와 테두리 사이의 여백
  • Border: 요소의 테두리
  • Margin: 요소와 외부 요소 사이의 여백

3. 박스 크기 계산 방식

  • 기본적으로 width와 height는 content 영역만을 의미
  • box-sizing: content-box: 기본값 (content만 width/height 포함)
  • box-sizing: border-box: padding과 border를 포함한 크기로 계산
  • 대부분의 실무에서는 border-box를 선호 (예상 가능한 박스 크기 설정 가능)

4. 요소 크기 고정과 overflow

  • 높이 또는 너비를 고정하면 콘텐츠가 넘칠 수 있음
  • overflow 속성으로 넘치는 콘텐츠 처리 방식 지정
  • 주요 값:
    • visible: 넘친 콘텐츠 표시 (기본값)
    • hidden: 넘친 콘텐츠 잘라냄
    • scroll: 항상 스크롤 표시
    • auto: 콘텐츠 넘칠 때만 스크롤 생성

5. 방향별 padding / margin 단축 표기법

  • padding: A; → 네 방향 모두 A
  • padding: A B; → 위아래 A, 좌우 B
  • padding: A B C; → 위 A, 좌우 B, 아래 C
  • padding: A B C D; → 위 A, 오른쪽 B, 아래 C, 왼쪽 D (시계방향)

6. block 요소와 크기

  • block 요소는 기본적으로 width: 100% (부모 너비 채움)
  • height: auto일 경우, 콘텐츠의 높이에 따라 자동 증가
  • width가 고정된 block 요소를 가운데 정렬하려면 margin: 0 auto; 사용

7. inline-block 요소

  • inline 요소처럼 나란히 배치되지만, width / height 지정 가능
  • 이미지, 버튼, 폼 요소 등에 자주 사용

8. 참고 예시

  • border: 5px dashed green; → 테두리 두께 5px, 초록색, 점선
  • border-width: top right bottom left; → 방향별로 테두리 너비 지정
  • border-color: top right bottom left; → 방향별로 색상 지정
  • margin: 20px auto; → 위아래 20px, 좌우 자동 정렬 (중앙 정렬)

9.대표적인 Block 요소(display:block)

  • <div> : 레이아웃 용도로 가장 많이 사용되는 블럭
  • <p> : 문단(Paragraph), 텍스트 단락 표현
  • <h1> ~ <h6> : 제목 태그 (h1이 가장 크고 중요)
  • <ul>, <ol>, <li> : 목록 태그 (순서 없음/있음)
  • <header>, <footer> : 문서나 영역의 머리말, 꼬리말
  • <section>, <article>, <aside>, <nav> : 시맨틱 구조 태그
  • <table> : 표를 만들기 위한 블럭
  • <form>, <fieldset> : 입력 영역 그룹화
  • <hr> : 수평선 (구분선)
  • <blockquote> : 인용문 표현

10.대표적인 Inline-Block 요소

  • <img> : 이미지를 삽입하며, 기본적으로 inline-block
  • <input> : 사용자 입력 필드 (text, radio, checkbox 등)
  • <button> : 버튼 요소
  • <select> : 드롭다운 선택 목록
  • <textarea> : 여러 줄 입력 필드
  • <label> : 폼 요소에 대한 설명
  • <iframe> : 인라인 프레임 (다른 HTML 문서 삽입)
  • <video>, <audio> : 미디어 요소 (컨트롤러 포함)
  • <canvas> : 그래픽 그리기 위한 요소
  • 커스텀 스타일을 통해 display: inline-block;으로 지정한 모든 요소

11.Inline 요소의 박스 모델 특징

  • 줄 바꿈이 되지 않고, 옆으로 배치된다 (같은 라인에 나란히 정렬)
  • 컨텐츠의 크기만큼만 가로 영역을 차지한다
  • width와 height 속성은 무시된다 (지정해도 적용되지 않음)
  • padding과 margin 중 좌우(Left, Right)는 적용된다
  • padding과 margin 중 상하(Top, Bottom)는 공간에 영향을 주지 않거나 겹침 현상 발생
  • box-sizing 속성은 기본적으로 content-box이며, 실질적으로는 제한적 효과
  • 박스 모델을 완전히 적용하려면 display 속성을 inline-block 또는 block으로 변경해야 한다

12.대표적인 Inline 요소(box모델이 적용되지 않음)

  • <span> : 의미 없는 텍스트 구분용 인라인 컨테이너
  • <a> : 하이퍼링크 텍스트
  • <strong> : 중요 강조 (굵게)
  • <em> : 감정적 강조 (기울임)
  • <br> : 줄바꿈 (라인 브레이크)
  • <q> : 짧은 인라인 인용문 (큰 따옴표)
  • <abbr> : 약어(abbreviation) 정의
  • <cite> : 출처 표기
  • <code> : 인라인 코드(모노스페이스 폰트)
  • <mark> : 하이라이트(형광펜 표시)
  • <time> : 날짜/시간 표기
  • <sub>, <sup> : 아래 첨자 / 위 첨자
  • <b>, <i>, <u> : 단순한 굵기, 기울임, 밑줄 (의미 없음)
  • <address>: 문서 작성자의 연락처 정보, 블록 요소, 기울임, 바닥글이나 작성자 정보에 사용
  • <pre>: 포맷팅 보존된 코드 블록, 블록 요소, 고정폭 글꼴, 공백 유지, 여러 줄 코드에 사용
  • <kbd>: 키보드 입력 내용, 인라인 요소, 고정폭 글꼴, 입력 안내 예: Ctrl + S
  • <samp>: 프로그램 출력, 인라인 요소, 고정폭 글꼴, 예: 404 Not Found 출력

13.object-fit 속성 정리

  • 이미지나 비디오 같은 replaceable 요소의 콘텐츠가 박스 영역에 어떻게 맞춰질지 결정하는 CSS 속성
  • 주로 <img> 태그와 함께 사용하여 이미지 크기 조절과 잘림 여부를 제어
  • 주요 속성 값:
    • fill: 기본값, 콘텐츠를 박스 크기에 맞게 늘리거나 줄여 비율 무시, 왜곡 발생 가능
    • contain: 콘텐츠 비율 유지하며 박스 안에 모두 들어가도록 축소 또는 확대, 빈 공간 생길 수 있음
    • cover: 콘텐츠 비율 유지하며 박스를 꽉 채우도록 확대, 일부 콘텐츠는 잘릴 수 있음
    • none: 원본 크기 유지, 박스 크기와 맞지 않으면 잘릴 수 있음
    • scale-down: none 또는 contain 중 더 작은 크기로 조절
  • 예시: 
    img {
  width: 300px;
  height: 200px;
  object-fit: cover;
}
    이미지가 300x200 크기로 영역을 채우며, 비율 유지되지만 일부 잘릴 수 있음
  • 배경 이미지에 비슷한 효과를 주려면 background-size: cover; 사용

박스 모델 & display 속성 실습 과제

  • <div> 태그를 사용해 빨간색 배경의 block 박스를 만들고, 내부에 텍스트를 넣으세요.
    (조건: width 300px, height 100px, padding 20px, margin 20px auto)
  • <span> 태그를 사용해 텍스트 일부를 강조하세요.
    (조건: background-color: yellow; display 속성 변경 없이 inline 상태 유지)
  • <img> 태그를 inline-block 박스로 인식하게 만들고, 아래와 같이 스타일을 지정하세요.
    (조건: width: 150px; height: 150px; border-radius: 50%)
  • <a> 태그를 사용하여 링크 텍스트를 만들고, padding과 margin을 적용해 보세요.
    (조건: display는 inline, padding: 10px 20px, margin: 10px 0 적용 시 어떻게 동작하는지 확인)
  • <button> 태그를 사용하여 버튼을 만들고 inline-block 요소의 특징을 확인하세요.
    (조건: width, height를 자유롭게 지정하고, text-align과 padding도 조정해볼 것)
  • display 속성의 차이를 비교하기 위한 실험을 하세요.
    같은 콘텐츠를 <div>, <span>, <button>으로 만들어 각각 display 속성에 따른 배치를 비교하세요.
  • 하나의 <section> 안에 block 요소, inline 요소, inline-block 요소를 섞어서 정렬해 보세요.
    (조건: 각 요소에 border를 주고 박스 크기와 배치 방식이 어떻게 다른지 시각적으로 비교)
