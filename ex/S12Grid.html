CSS Grid 기초(쉬운 버전)

가장 자주 쓰는 개념만 간단히 익히고, 바로 눈으로 확인하는 데모

0) Grid는 언제 쓰나

한 줄 배치만 필요하면 Flex, 행/열 2차원 배치가 필요하면 Grid가 편합니다. 카드 목록, 페이지 레이아웃, 사이드바 구조에 특히 유용합니다.

1) Grid 시작하기: display, columns, gap

display: grid — “Grid를 쓰겠다”는 선언입니다.
grid-template-columns — 열(세로 트랙) 개수/크기를 정합니다.
gap — 칸과 칸 사이 간격입니다(가로/세로 동시).

1
2
3
4
.ex1{
  display: grid;                 /* Grid 활성화 */
  grid-template-columns: 1fr 1fr;/* 열 2개, 남은 공간을 1:1 분배 */
  gap: 12px;                     /* 셀 사이 간격 */
}

2) 고정 + 유동 혼합: 120px 1fr 120px

1fr — 남은 공간을 “비율”로 나눕니다.
120px — 고정 너비 칼럼입니다. 고정과 fr을 섞어 자주 씁니다.

3) repeat()로 간단히 N열 만들기

repeat(3, 1fr) — 같은 패턴을 3번 반복합니다. 코드가 짧아집니다.

.ex3{
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 1fr 열 3개 */
  gap: 10px;
}

4) 최소~최대 크기: minmax() + auto-fit

minmax(최소, 최대) — 칼럼이 너무 좁아지지 않게 최소폭을 보장합니다.
auto-fit — 가능하면 칼럼을 접었다 펴서 빈 칸 없이 딱 맞게 채웁니다.

.ex4{
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(160px, 1fr));
  gap: 10px;
}

5) 아이템 위치 지정: grid-column / grid-row

grid-column: 시작라인 / 끝라인 — 가로로 몇 칸 차지할지 지정합니다.
grid-row: 시작라인 / 끝라인 — 세로로 몇 칸 차지할지 지정합니다.

.ex5{
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(2, 80px);
  gap: 8px;
}
.ex5 .wide{ grid-column: 1 / 3 } /* 1~3 라인(= 2칸) */
.ex5 .tall{ grid-row: 1 / 3 }   /* 1~3 라인(= 2칸) */

6) 정렬: place-items(전체), place-self(개별)

place-items — 모든 칸 안의 내용 정렬(가로+세로 단축).
place-self — 특정 아이템 한 개만 따로 정렬.

center
start end
center
center
center
center
.ex6{
  display: grid;
  grid-template-columns: repeat(3, 140px);
  grid-template-rows: repeat(2, 90px);
  gap: 8px;
  place-items: center;      /* 각 칸의 내용 가운데 */
}
.ex6 .cell:nth-child(2){
  place-self: start end;    /* 이 칸만 위-오른쪽 */
}

7) 이름으로 배치: grid-template-areas

grid-template-areas — 레이아웃을 그림처럼 선언합니다.
grid-area — 각 박스를 해당 이름에 연결합니다.
미디어쿼리로 이름 배열만 바꿔도 손쉽게 재배치됩니다.

Header
Nav
Main
Footer
.ex7{
  display: grid;
  grid-template-columns: 160px 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "hd hd"
    "nv mn"
    "ft ft";
  gap: 8px;
}
.hd{ grid-area: hd } .nv{ grid-area: nv }
.mn{ grid-area: mn } .ft{ grid-area: ft }

@media (max-width:700px){
  .ex7{
    grid-template-columns: 1fr;
    grid-template-areas:
      "hd"
      "nv"
      "mn"
      "ft";
  }
}

끝내기 전에 체크

display: grid를 썼는가? gap으로 간격을 줬는가?
columns/rows, repeat(), fr, minmax()의 의미가 헷갈리지 않는가?
아이템 배치(grid-column/row)와 정렬(place-items/self)을 구분할 수 있는가?
