과제 1: 회원가입(Sign-up) 페이지

  • 목표
    Bootstrap Form 컴포넌트와 유틸리티 클래스로 반응형 회원가입 폼 구현
  • 필수 요소
    이름, 이메일, 비밀번호, 비밀번호 확인, 약관 동의 체크박스, 가입 버튼
  • 레이아웃
    모바일 col-12, 태블릿 col-md-8, 데스크톱 col-lg-6 중앙 정렬
    (컨테이너 + row + col, mx-auto 활용)
  • UI 디테일
    - 폼 상단에 페이지 타이틀
    - 하단에 “로그인 페이지로 이동” 링크
    - 제출 시 버튼 내 로딩 스피너 표시
  • 반응형 포인트
    입력 그룹 사이 간격은 gap-* 또는 mb-* 유틸 활용,
    여백은 py-5 이상 지정
