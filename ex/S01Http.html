HTML 수업 1일차

1. 인터넷이란?

인터넷은 전 세계의 컴퓨터들이 서로 연결된 네트워크입니다. 사용자는 웹 브라우저를 통해 전 세계의 웹 서버에 접속할 수 있습니다.

  • IP 주소를 통해 컴퓨터를 식별
  • DNS를 통해 도메인 이름을 IP 주소로 변환
  • 웹 브라우저는 서버에 요청(Request)을 보내고, 서버는 응답(Response)을 보냄

2. HTTP란?

HTTP(HyperText Transfer Protocol)는 웹에서 데이터를 주고받기 위한 통신 규약입니다.

  • 요청(Request): 클라이언트가 서버에 보내는 메시지
  • 응답(Response): 서버가 클라이언트에게 보내는 메시지

3. HTML이란?

HTML(HyperText Markup Language)은 웹 페이지를 구성하는 기본 언어입니다.

  • 웹상 주고 받는 데이터로 마크업으로 뼈대를 구성함
  • 태그(Tag)를 사용하여 문서의 구조를 정의함
  • 브라우저가 해석해서 사용자에게 시각적으로 보여줌
  • 정적인 문서이지만 CSS, JavaScript와 함께 동적인 페이지를 만들 수 있음

4. URL이란?

URL(Uniform Resource Locator)은 인터넷 상의 자원이 위치한 주소를 의미합니다. 사용자가 웹 브라우저에 입력하는 주소이며, 웹 페이지, 이미지, 파일 등 모든 자원에 접근하기 위해 사용됩니다.

  • 리소스의 위치와 접근 방법(프로토콜)을 함께 명시
  • 형식: 프로토콜://도메인:포트/경로?쿼리#해시
  • 예: https://www.example.com:443/page?search=apple#section1
  • 주요 구성 요소:
    • 프로토콜: https, http, ftp 등
    • 도메인: www.example.com
    • 포트: 생략 가능 (기본 포트는 자동 사용)
    • 경로: 웹 서버의 파일이나 페이지 경로
    • 쿼리: ? 이후의 요청 매개변수 (key=value)
    • 해시: # 이후는 페이지 내부 위치로 이동

5. XML이란?

XML(eXtensible Markup Language)은 데이터를 구조화하여 저장하고 전달하기 위한 마크업 언어입니다. 주로 시스템 간의 데이터 교환, 설정 파일 등에 사용됩니다.

  • 태그를 사용하여 데이터의 구조와 의미를 표현
  • 사용자가 직접 태그를 정의할 수 있어 확장 가능함
  • HTML처럼 사람이 읽을 수 있지만, 주 목적은 데이터 저장과 전송
  • 트리 구조로 이루어져 있어 계층적 데이터 표현에 적합
  • 다양한 플랫폼과 언어에서 처리 가능 (예: Java, Python, JavaScript 등)
  • 예시: 
    <person>
    <name>홍길동</name>
    <age>30</age>
</person>

6. HTML 태그의 종류

HTML 태그는 크게 블럭 요소와 인라인 요소로 나눌 수 있습니다.

  • 블럭 요소(Block Element)
    • 한 줄 전체를 차지하며 위아래로 줄 바꿈이 생깁니다. (웹상의 UI가 스크롤로 이루어져있어서)
    • 주로 레이아웃 구조나 큰 덩어리를 구성할 때 사용됩니다.
    • 블럭 요소 안에 다른 블럭 요소나 인라인 요소를 포함할 수 있습니다.
  • 인라인 요소(Inline Element)
    • 줄 바꿈 없이 한 줄 안에 나란히 배치됩니다.
    • 주로 텍스트의 일부분이나 짧은 요소를 꾸밀 때 사용됩니다.
    • 인라인 요소 안에는 블럭 요소를 넣을 수 없습니다.

7. 블럭 요소 예시

  • <div> : 구역 또는 레이아웃 분할
  • <p> : 문단
  • <h1> ~ <h6> : 제목
  • <ul>, <ol> : 리스트(순서 없는/있는)
  • <li> : 리스트 항목
  • <section>, <article>, <nav>, <header>, <footer> 등
  • <form> : 양식 요소

8. 인라인 요소 예시

  • <span> : 인라인 구역
  • <a> : 하이퍼링크
  • <strong>, <em> : 강조
  • <img> : 이미지 삽입
  • <input>, <label>, <select> : 폼 관련 요소들
  • <br> : 줄 바꿈
  • <code> : 코드 조각
