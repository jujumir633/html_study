HTML 요소의 속성 정리

속성이란?

  • HTML 요소에 부가 정보를 제공함
  • 항상 이름="값" 형식으로 작성
  • 시작 태그 내부에 작성

속성 예시

  • <a href="https://example.com"> : 링크 주소
  • <img src="image.jpg" alt="설명"> : 이미지 경로와 대체 텍스트
  • <input type="text" value="기본값"> : 입력값 설정
  • <button disabled> : 버튼 비활성화
  • <div title="툴팁"> : 마우스를 올렸을 때 설명 표시

공통 속성(Global Attributes)

  • id : 고유 식별자
  • class : 클래스 이름 (CSS, JS에서 사용)
  • style : 인라인 스타일 지정
  • title : 툴팁 제공
  • hidden : 화면에서 숨김 처리
  • data-* : 사용자 정의 데이터 속성

Input 요소의 주요 속성

  • type : 입력 필드의 유형 설정 (text, password, email 등)
  • value : 초기 입력값 설정
  • placeholder : 입력 전 표시되는 안내 텍스트
  • name : 폼 전송 시 사용되는 이름
  • required : 필수 입력 필드로 지정
  • readonly : 읽기 전용으로 설정
  • disabled : 비활성화 상태로 설정
  • maxlength : 입력 가능한 최대 글자 수 제한
  • autocomplete : 자동 완성 기능 설정 (on/off)

예시

  • <input type="text" placeholder="이름을 입력하세요">
  • <input type="password" required>
  • <input type="email" value="user@example.com" readonly>
  • <input type="text" disabled value="수정 불가">
  • <input type="text" maxlength="10">
