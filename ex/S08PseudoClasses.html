의사 클래스(Pseudo-class)

의사 클래스(Pseudo-class)란?

  • HTML 요소의 특정 **상태나 조건**에 따라 스타일을 적용할 수 있도록 해주는 CSS 선택자입니다.
  • 예를 들어, 마우스를 올렸을 때(`:hover`), 체크박스가 선택됐을 때(`:checked`) 등에만 적용되는 스타일을 만들 수 있습니다.
  • 콜론(:)으로 시작하며, HTML 구조나 상태에 따라 요소를 선택합니다.

:hover

  • 마우스를 요소 위에 올렸을 때 적용
  • 예시: 
    
button:hover {
  background-color: gold;
}

:active

  • 요소를 클릭하는 순간 (눌리는 중) 적용
  • 예시: 
    
a:active {
  color: red;
}

:focus

  • 요소에 포커스가 갔을 때 (ex. input 클릭 시)
  • 예시: 
    
input:focus {
  border-color: blue;
}

:checked

  • 체크박스나 라디오 버튼이 선택되었을 때 적용
  • 예시: 
    
input:checked + label {
  color: green;
}

:disabled / :enabled

  • 폼 요소가 비활성화 / 활성화 상태일 때 적용
  • 예시: 
    
button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}

:nth-child(n)

  • 부모의 n번째 자식 요소에 적용
  • 예시: 
    
li:nth-child(2) {
  color: orange;
}

:first-child / :last-child

  • 부모 요소의 첫 번째 / 마지막 자식 요소에 스타일 적용
  • 예시: 
    
p:first-child {
  font-weight: bold;
}

:not(selector)

  • 특정 요소를 제외한 나머지에 스타일 적용
  • 예시: 
    
div:not(.active) {
  opacity: 0.5;
}

:is(...)

  • 여러 선택자를 묶어서 하나로 간결하게 작성
  • 예시: 
    
:is(h1, h2, h3) {
  font-family: sans-serif;
}

:empty

  • 내용이 완전히 없는 요소에만 스타일 적용 (공백도 없어야 적용됨)
  • 예시: 
    
div:empty {
  background-color: lightgray;
}

:root

  • HTML 문서의 최상위 요소(html)를 선택
  • CSS 변수 정의에 자주 사용
  • 예시: 
    
:root {
  --main-color: royalblue;
}
