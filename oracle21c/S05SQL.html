SQL이란

SQL 개요

  • SQL(Structured Query Language)은 관계형 데이터베이스를 다루는 언어
  • 데이터 정의, 조작, 조회, 제어 모두 SQL로 수행
  • ANSI와 ISO에 의해 제정된 국제 표준 언어
  • 오라클, MySQL, PostgreSQL, SQL Server 등 모든 DBMS에서 공통 사용

데이터 기본 연산 – CRUD

  • 데이터베이스의 네 가지 기본 동작
  • Create (등록): INSERT
  • Read (조회): SELECT
  • Update (수정): UPDATE
  • Delete (삭제): DELETE

SQL 명령어 분류

  • DDL (Data Definition Language): 데이터 정의
    • CREATE, ALTER, DROP, TRUNCATE
    • 테이블, 뷰, 인덱스 등 구조 정의
  • DML (Data Manipulation Language): 데이터 조작
    • INSERT, UPDATE, DELETE
    • 실제 데이터 등록/수정/삭제
  • DQL (Data Query Language): 데이터 조회
    • SELECT
    • 데이터 검색 및 조건, 정렬, 집계 수행
  • TCL (Transaction Control Language): 트랜잭션 제어
    • COMMIT, ROLLBACK, SAVEPOINT
    • 작업의 완료와 취소를 관리
  • DCL (Data Control Language): 권한 제어
    • GRANT, REVOKE
    • 사용자 권한 부여 및 회수

DQL의 위치에 대한 관점

  • ANSI/ISO 표준: SELECT는 DML에 포함
  • 교육/실무: SELECT의 비중이 커서 DQL로 따로 분리해 설명하기도 함
  • 정리: SELECT는 DML의 일부지만, 학습할 때는 DQL로 구분하면 이해가 쉬움
